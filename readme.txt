== Description ==

The default Archives widget simply lists links to the monthly archives. This plugin gives you a new widget that groups the months by year with a jQuery accordion effect.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
