== Description ==

This plugin adds a \"custom archives widget\" to your WordPress installation that allows you to display links to archive pages grouped by year.

It has an accordion effect to reveal the list of months under the currently hovered year link

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
